# JSK 灵魂测试网站规格文档

## 1. 项目概述

- **项目名称**: JSK SOUL TEST
- **项目类型**: 人格测试类单页Web应用
- **核心功能**: 基于四位真实人物原型设计的个性化人格测试，展示四种独特的灵魂类型
- **目标用户**: 网站创始人及其朋友们的纪念网站

## 2. 视觉风格

### 2.1 整体美学 — 潮流个性风格
- **色调**: 深色背景 + 霓虹渐变（粉色#ff6b9d、紫色#c084fc、青色#22d3ee）
- **字体**:
  - 标题: ZCOOL KuaiLe（个性手写体）
  - 正文: Noto Sans SC（现代无衬线）
- **效果**: 霓虹光晕、网格背景、渐变发光按钮
- **动效**: 流畅滑入动画、悬停光效

### 2.2 色彩方案
```
背景色（Dark）: #0a0a0f
卡片色（Card）: #12121a
主色（Primary）: #ff6b9d（霓虹粉）
次色（Secondary）: #c084fc（霓虹紫）
强调色（Accent）: #22d3ee（霓虹青）
文字色（Text）: #e2e8f0
边框色（Border）: #1e1e2e
```

### 2.3 页面结构
- **首页**: 大标题 + 渐变logo + 氛围文案 + 发光按钮
- **测试页**: 进度条 + 维度标签 + 题目卡片 + 选项列表
- **结果页**: 四类分析展示（性格底色/情感态度/人生态度/关键词）
- **管理页**: 题目列表 + 新增/编辑/删除功能（口令保护）

## 3. 四种人格类型

| 类型代码 | 名称 | 原型人物 | 关键词 |
|---------|------|---------|--------|
| A | 追光者 | 邓宏锦 | 高共情、独立思考、随性洒脱、清醒恋爱 |
| B | 破晓者 | 金熙 | 真诚坦荡、氛围maker、乐观担当、义气为先 |
| C | 深海者 | 岳思语 | 内敛深沉、默默承担、外冷内热、回避依赖 |
| D | 漫游者 | 陈舒窈 | 掌控放手、情感漠视、自私真实、通透矛盾 |

### 3.1 结果分析维度
- **性格底色**: 原生性格特质形成
- **情感态度**: 对爱情、友情的看法与处理方式
- **人生态度**: 对生活、未来规划的态度
- **关键词**: 6个核心特质标签

## 4. 题目维度

测试包含四个维度，共20道题：

| 维度 | 标签 | 说明 |
|-----|------|------|
| EMOTION | 情感 | 感情态度、恋爱观、人际关系处理 |
| LIFE | 人生 | 生活态度、未来规划、人生选择 |
| VALUE | 价值观 | 金钱观、成熟观、社会观 |
| SELF | 自我 | 自我认知、性格倾向、内在外在 |

每道题两个选项，分别倾向不同人格类型。

## 5. 功能模块

### 5.1 测试模块
- 20道趣味选择题
- 维度标签显示（情感/人生/价值观/自我）
- 霓虹风格进度条
- 上一题/下一题导航
- 最后一题显示"查看结果"

### 5.2 结果展示模块
- 显示人格类型代码（A/B/C/D）和名称
- 四维度深度分析
- 6个关键词标签
- 重新测试按钮

### 5.3 题目管理模块
- 管理口令: `jskbti2024`
- 查看所有题目（按维度分类）
- 新增题目（设置维度 + 选项 + 倾向）
- 编辑现有题目
- 删除题目
- 数据存储在 localStorage

## 6. 技术方案

- **前端**: 纯HTML + CSS + JavaScript（单页应用）
- **数据存储**: localStorage 存储题目数据
- **无需后端**: 所有功能前端实现
- **响应式**: 适配移动端和桌面端

## 7. 验收标准

1. 深色潮流风格界面正常显示
2. 测试流程完整（20道题）
3. 结果正确对应四种人格类型之一
4. 四维度分析展示完整
5. 题目管理功能正常（需正确口令）
6. 移动端显示正常
